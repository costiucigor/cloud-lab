Key Commands

Create a VM

gcloud compute instances create my-vm \ --zone=europe-central2-a \ --machine-type=e2-micro \ --image-family=debian-12 \ --image-project=debian-cloud \ --tags=http-server

Create a Storage Bucket

gsutil mb -l europe-central2 gs://my-bucket-name # Upload files gsutil cp index.html gs://my-bucket-name/ # Make public gsutil iam ch allUsers:objectViewer gs://my-bucket-name

Static Website Hosting

# Set main page gsutil web set -m index.html gs://my-bucket-name # Access at: https://storage.googleapis.com/my-bucket-name/index.html
