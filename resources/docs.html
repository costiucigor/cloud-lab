Module 1: Google Apps Script

Setup Instructions

  1. Go to script.google.com
  2. Create new project
  3. Copy Code.gs content
  4. Create HTML files: File ‚Üí New ‚Üí HTML
  5. Update appsscript.json (Project Settings ‚Üí Show manifest)
  6. Create a Google Sheet and copy its ID
  7. Replace YOUR_SPREADSHEET_ID_HERE in code
  8. Run function and authorize

üí° Tip

For form triggers, go to Triggers ‚Üí Add Trigger ‚Üí Select "onFormSubmit" ‚Üí On form submit

Modules 2-4: GCP Infrastructure

Running Shell Scripts

  1. Open Google Cloud Shell
  2. Upload script files or paste content
  3. Make executable: chmod +x script.sh
  4. Run: ./script.sh
  5. Wait for resources to be created
  6. Access via external IP shown in output

‚ö†Ô∏è Cleanup

Always run the delete script when done to avoid charges!

Module 5: PaaS & AI APIs

App Engine Deployment

  1. Enable App Engine API in GCP Console
  2. Run gcloud app create --region=europe-west1
  3. Deploy: gcloud app deploy
  4. View: gcloud app browse

AI APIs Setup

  1. Enable APIs: Vision, Translation, Text-to-Speech, Natural Language
  2. Create Service Account with appropriate roles
  3. Download JSON key
  4. Set GOOGLE_APPLICATION_CREDENTIALS environment variable
  5. Install dependencies: npm install
  6. Run server: node server.js

Module 6: Machine Learning

Vertex AI Training

  1. Upload flags.csv to Cloud Storage bucket
  2. Go to Vertex AI ‚Üí Datasets ‚Üí Create
  3. Select Tabular, Classification
  4. Import from Cloud Storage
  5. Train with AutoML (1 node hour minimum)
  6. Evaluate metrics after training
  7. Deploy to endpoint for predictions

Python Visualization

# Install dependencies pip install pandas matplotlib # Run visualization python visualize-data.py

Module 7: Load Balancing

Full Setup

  1. Upload full-setup.sh to Cloud Shell
  2. Edit USERNAME variable in script
  3. Run: ./full-setup.sh
  4. Wait 5-10 minutes for all resources
  5. Get Load Balancer IP from output
  6. Test by refreshing page (different hostname each time)

üí° Testing Load Balancer

Run for i in {1..10}; do curl http://LB_IP/; done to see traffic distributed across VMs

API Reference (This Site)

POST /api/vision/labels

Detect labels in an image

POST /api/translate

Translate text
